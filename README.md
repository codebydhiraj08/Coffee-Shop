# Brewed Bliss Café Website

A modern, responsive website for Brewed Bliss Café in Pune, featuring a beautiful design with coffee-themed aesthetics.

## Features

- **Hero Section**: Video background with smooth crossfade animation
- **Menu Display**: Interactive menu cards with hover effects
- **About Section**: Information about the café with feature highlights
- **Testimonials**: Customer reviews with star ratings
- **Contact Form**: Reservation system
- **Responsive Design**: Works on all devices
- **Smooth Animations**: Scroll reveals and interactive elements

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- JavaScript (ES6+)
- Google Fonts (Playfair Display, Cormorant Garamond, DM Sans)

## Project Structure

    └── ... (other images)

## Setup

1. Clone or download the project
2. Open `index.html` in a modern web browser
3. For development, use Live Server extension in VS Code

## Development

- **Live Server**: Install the Live Server extension in VS Code for local development
- **Formatting**: Prettier is configured for code formatting
- **Extensions**: Recommended VS Code extensions are installed

## Browser Support

- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+
